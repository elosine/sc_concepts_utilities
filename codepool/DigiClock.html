DigiClock Digital timer


Inherits from: Object


A digital timer GUI.


See also: SCView, GUI, SCScrollTopView, Pen


Creation / Class Methods


*new(window,scale, displayColor,moveX,moveY,font)

Creates a new DigiClock instance. Currently you can run more than one clock at once, but if you stop or reset one DigiClock they all stop.

window - Parent window in which to embed the clock. If nil, the clock will be created within its own window.

scale - This is a multiplier for the size of the DigiClock window. The Default is 1.

displayColor - Color of the number display. The Default is cyan.

moveX - X coordinant position.

moveY - Y coordinant position.

fontÂ  - The font used for the clock; must be given as a string. The Default is "DS-Digital" which must be downloaded first to be used. Web Site: http://www.dailyfreefonts.com/fonts/infos/38-DS-Digital.html

Instance Methods


start

Starts the DigiClock timer.


stop

Pauses the DigiClock, but does not reset it


reset

Stops and resets the DigiClock timer.

close

Stops and closes the DigiClock timer.


Examples


~clock = DigiClock.new;

var window,clocks,view;

window.front;
